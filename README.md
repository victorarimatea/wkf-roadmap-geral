## O que é este workflow

Processo semanal de revisão estratégica do ROADMAP do ecossistema DTD/SETIS/SES-DF.
Transforma o ROADMAP de lista de tarefas em instrumento estratégico vivo — capaz de
registrar o que foi construído, comunicar para onde o projeto vai, e processar ideias
emergentes antes que se percam.

## Ciclo de execução

**Fixo:** sextas-feiras à tarde — uma vez por semana
**Extraordinário:** quando a staging area acumular mais de 10 itens Tipo 1
ou qualquer ideia Tipo 2 com mais de 7 dias sem processamento
